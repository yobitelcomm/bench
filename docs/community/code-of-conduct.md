# Code of conduct

InferenceBench follows the [Contributor Covenant](https://www.contributor-covenant.org/) code of conduct. The canonical text lives in the repository at [CODE_OF_CONDUCT.md](https://github.com/yobitelcomm/bench/blob/main/CODE_OF_CONDUCT.md).

## In short

- Be respectful.
- Assume good faith.
- Critique the work, not the person.
- Take disagreements to the issue tracker, not to private channels.

## Reporting

Report violations by opening a private issue on the GitHub repository or emailing the maintainers (contact on the [GitHub org page](https://github.com/yobitelcomm)). Reports are confidential.

## Scope

The code of conduct applies to all project spaces: GitHub issues and PRs, the documentation site, any community channels, and in-person events tied to the project.
